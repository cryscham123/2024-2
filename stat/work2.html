a

20대: 15% * 58% = 8.7% 30대: 24% * 52% = 12.48% 40대: 26% * 44% = 11.44% 50대: 20% * 40% = 8% 60대 이상: 15% * 35% = 5.25% 전체 지지율 = 8.7% + 12.48% + 11.44% + 8% + 5.25% = 45.87% 따라서 이 후보의 지지율은 약 45.87%입니다.

b

이는 베이즈 정리를 사용하여 계산할 수 있습니다. P(연령대|지지) = P(지지|연령대) * P(연령대) / P(지지) 20대: (58% * 15%) / 45.87% = 18.97% 30대: (52% * 24%) / 45.87% = 27.21% 40대: (44% * 26%) / 45.87% = 24.94% 50대: (40% * 20%) / 45.87% = 17.44% 60대 이상: (35% * 15%) / 45.87% = 11.44%

9

주어진 정보:

인구의 20%가 흡연자 흡연자의 폐암 사망 확률은 비흡연자의 10배 전체 폐암 사망 확률: 0.028

Let x = 비흡연자의 폐암 사망 확률 Then, 10x = 흡연자의 폐암 사망 확률 0.028 = 0.8x + 0.2(10x) 0.028 = 0.8x + 2x 0.028 = 2.8x x = 0.01 따라서:

비흡연자의 폐암 사망 확률: 0.01 (1%) 흡연자의 폐암 사망 확률: 0.1 (10%)

폐암 사망자가 흡연자일 확률 (베이즈 정리 사용): P(흡연|폐암) = P(폐암|흡연) * P(흡연) / P(폐암) = 0.1 * 0.2 / 0.028 = 0.714 (71.4%)
