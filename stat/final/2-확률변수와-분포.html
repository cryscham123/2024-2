확률변수

sample space의 원소를 실수로 대응시키는 함수 - 이산확률변수: 확률변수가 취할 수 있는 값이 유한개 또는 무한개이지만 셀 수 있는 경우 - 연속확률변수: 확률변수가 취할 수 있는 값이 실수의 구간이고 셀 수 없는 경우 이산 표본공간 -> 이산 확률변수 연속 표본공간 -> 연속 확률변수 연속 표본공간 -> 이산 확률변수

확률 실험 -> 표본공간 -> 확률변수 -> 확률분포

확률 분포

이산확률분포

확률 질량 함수(pmf): P(X=x) = f(x) => X가 x일 확률 - 기하분포: 성공확률 p인 베르누이 시행을 독립적으로 반복했을 때 첫 번째 성공이 나타날 때까지의 시행횟수

연속확률분포

  • 확률 밀도 함수(pdf): \(\int{f(x)}dx = 1\)

  • \(\int[a,b]{f(x)}dx = P(a ≤ x ≤ b)\) => x가 a와 b사이에 있을 확률

  • P(X=x) = 0 (연속형 데이터여서 특정값을 가질 확률은 0)

  • f(x)는 1보다 큰 값을 가질 수 있음

  • 누적분포함수(cdf): \(F(x) = P(X ≤ x)\) (이산확률분포와 연속확률분포 모두에 적용)
