1장- 집합, 실수의 성질

기초공학수학
Published

September 2, 2024

정의

  • 집합: 명확하게 식별할 수 있는 대사의 모음
  • 원소: 집합을 구성하는 대상
  • 집합의 표현
    • 원소 나열법: 집합의 원소를 모두 나열하는 방법
      ex) A = {1, 2, 3, 4, 5}
    • 조건 제시법: 집합의 원소가 만족해야 하는 조건을 나열하는 방법
      ex) A = {x | x ⋴ 𝕟, 1 ≤ x ≤ 5}
  • 부분집합: 집합 A의 모든 원소가 집합 B의 원소이면 집합 A는 집합 B의 부분집합이다.
    ex) A = {1, 2, 3}, B = {1, 2, 3, 4, 5}
    A ⊆ B
  • 진부분집합: A ⊆ B 이면서 A ≠ B인 경우
  • 공집합: 어떤 집합의 원소도 포함하지 않는 집합
  • 집합 연산
    • 합집합: A ∪ B = {x | x ⋴ A 또는 x ⋴ B}
    • 교집합: A ∩ B = {x | x ⋴ A 그리고 x ⋴ B}
    • 차집합: A - B = {x | x ⋴ A 그리고 x ⋏ B}
    • A x B = {(a, b) | a ⋴ A 그리고 b ⋴ B}
    • 여집합: A’ = {x | x ⋴ U 그리고 x ⋏ A} (U: 전체집합)

실수의 성질

유계집합

  • 위로 유계: A ⊆ R의 모든 원소 x에 대하여 x <= c인 c ∈ R이 존재
  • 아래로 유계: A ⊆ R의 모든 원소 x에 대하여 x >= d인 d ∈ R이 존재
  • 상계: 위로 유계인 집합에서 c
  • 하계: 아래로 유계인 집합에서 d
  • 상한: 상계 중에서 가장 작은 원소 (“sup A”)
  • 하한: 하계 중에서 가장 큰 원소 (“inf A”)

상한과 하한의 성질

  • 상한과 하한은 유일하다
  • 존재하지 않을 수 있다
  • 해당 집합의 원소일 필요가 없다.
